# 🔍 Source Type & License Guide

### Required Information by Type

**Data**
- Include license (CC-BY, etc.)
- Date accessed
- Note if you removed **PII (Personally Identifiable Information)**

**Research**
- Provide full citation  
  *(Author, Year, Title)*
- Include **DOI link**
- Provide **fair use justification** if applicable

**Code**
- Package name
- Version number
- License *(MIT, Apache, etc.)*
- Link to repository

**Visuals**
- Name the creator/photographer
- Include specific license *(Unsplash, Pexels, etc.)*

**Quotes**
- List the person’s **name and role**
- Confirm **"Consent received"**

**✅ Permissive (Safe)**  
- MIT

**⚠️ Restrictive (Use Caution)**  
- GPL *(requires your code to be open source)*  
- CC-BY-NC *(Non-Commercial — prize money may conflict)*  

**❌ Avoid**
- "All Rights Reserved"
- Unknown sources
- Images with watermarks
